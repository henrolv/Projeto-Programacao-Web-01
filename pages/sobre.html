Trabalho Programação Web 1

O projeto consiste no desenvolvimento de um site temático com, no mínimo, quatro páginas interconectadas em HTML, estilizadas por meio de um arquivo CSS externo. O objetivo é consolidar os conceitos de HTML e CSS, criando uma estrutura funcional e esteticamente agradável, que permita uma navegação fluida entre as páginas. Entre os requisitos técnicos, está a utilização de ao menos uma webfont, por exemplo, do Google Fonts, além de uma imagem em cada página e a adição de um favicon, que aparecerá na aba do navegador.

A equipe, composta por até quatro pessoas, deverá organizar o projeto em um repositório público no GitHub, facilitando a colaboração e a publicação no GitHub Pages. O site completo, juntamente com o repositório contendo o código-fonte, deve ser entregue até o dia 27/09/2024, permitindo que os avaliadores possam acessar tanto o produto final quanto o código utilizado no desenvolvimento.
